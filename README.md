## Inception

### Usage

Create an `.env` file in `srcs` directory with the following content, filling values for the variables:
```
MARIADB_DB=
MARIADB_USER=
MARIADB_PW=
MARIADB_ROOT_PW=
WORDPRESS_ADMIN_USER=
WORDPRESS_ADMIN_PW=
WORDPRESS_ADMIN_EMAIL=
WORDPRESS_TEST_USER=
WORDPRESS_TEST_PW=
WORDPRESS_TEST_EMAIL=
DOMAIN_NAME=
```
